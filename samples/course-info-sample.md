# 입력 샘플 — 강의 기본정보

`syllabus-design` 스킬의 입력 예시다. (HTML 프로토타입 STEP1 폼 필드에 대응)

| 항목 | 내용 |
|---|---|
| **과목명** | 데이터 기반 의사결정 입문 |
| **학문 분야** | 경영학 / 데이터 리터러시 (교양·전공기초) |
| **수강 대상** | 학부 3학년 (전공 무관, 통계 미이수자 다수) |
| **학점 / 시수** | 3학점, 주 3시간 (강의 2 + 실습 1) |
| **총 주차** | 15주 |
| **강의 방식** | 혼합(블렌디드) — 개념은 사전 동영상, 대면은 실습·토론 |
| **주요 내용·주제** | 데이터 리터러시 개념, 기술통계와 시각화, 표본과 추정, A/B 테스트, 상관 대 인과, 회귀의 직관, 대시보드로 의사결정 커뮤니케이션, 데이터 윤리 |
| **수강생 특성** | 전공 기초만 이수, 통계·프로그래밍 배경 이질적. 일부 창업·마케팅 실무 관심 학생 포함 |
| **평가 선호·수업 철학** | 과정 중심 평가 40%(주차별 실습·성찰 저널). 결과물 암기보다 "데이터로 의사결정을 정당화하는 능력" 중시. 생성형 AI는 데이터 해석 보조로 조건부 허용(출처·프롬프트 기록 의무) |

## 사용 예
```
/syllabus-design
위 강의 기본정보로 강의계획서를 설계해 주세요. (samples/course-info-sample.md 첨부)
```
→ ABCD 학습목표 + 15주차 일정표 + 평가 구성표 + 커리큘럼 맵이 생성된다.
