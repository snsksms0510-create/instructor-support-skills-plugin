# Canva 중심 도구 매핑

사용자 유형별로 제품이 갈리며 **요금제가 자동화 범위를 결정**하므로, 매핑 전 반드시 티어·권한을 확인한다.

## 강의안 활동(TLA) → Canva 기능

| 강의안 활동(TLA) | 대응 Canva 기능 |
|---|---|
| 강의 전달 | Presentations (설문·퀴즈·데이터 시각화 내장) |
| 브레인스토밍·협업 | Whiteboards |
| 배포 자료·과제 안내 | Docs |
| 데이터·연구 시각화 | Sheets (Magic Insights·Charts) |
| 영상 강의·플립러닝 | Video, Doc을 Presentation·Video로 변환 |
| 학생 산출물 과제 | LMS 연동 과제 |

## 티어·권한 주의
- **자동 생성(Autofill)**은 Campus/Enterprise 티어 + 관리자급 권한 전제. 무료·Pro·일반 교수자 계정이면 **수동 워크플로**로 분기한다.
- 자동 생성 시 **색상·글꼴은 변경 불가** 전제로 템플릿을 설계한다(API로 바꿀 수 있는 것은 텍스트·이미지·차트뿐).
- Canva가 약한 영역(실시간 형성평가·비동기 영상 토론)은 특화 도구로 보완한다(→ `tool-catalog.md`).
- 기관 보유 도구·LMS 연동·접근성을 우선 점검한다.
