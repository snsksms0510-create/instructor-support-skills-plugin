# 도구 카탈로그 (활동유형별 대표 도구)

도구 선택은 **활동 학습목표(인지수준)**가 결정한다. 도구를 먼저 정해 활동을 끼워 맞추지 않는다.

## 활동유형별 대표 도구

| 활동 유형 | 대표 도구 |
|---|---|
| 실시간 참여·형성평가 | Mentimeter, Poll Everywhere, Kahoot!, Socrative |
| 인터랙티브 강의·플립러닝 | Nearpod, Pear Deck, Edpuzzle |
| 협업·브레인스토밍 | Padlet, Miro, Google Workspace(Docs·Slides) |
| 비동기 영상 토론 | Flip |
| 글쓰기 피드백 | Google Docs(제안·댓글), Loom(영상 피드백) |
| 학습 허브(LMS) | Canvas, Moodle, Blackboard, Brightspace(D2L) |

## 도구 선택 점검 4문항
1. 활동 학습목표(인지수준)에 맞는가
2. 기관 LMS와 연동되는가
3. 학생이 새 계정 없이 쓸 수 있는가(도입 장벽)
4. 접근성(WCAG)·개인정보(FERPA 등) 요건을 충족하는가

> 도구 수를 늘리는 것 자체가 목적이 되어서는 안 되며, 학생이 익힐 도구는 최소화한다.

## LMS 공통 기능군
사용자의 LMS를 먼저 확인해 제품 용어로 제시하되, 공통 기능군으로 안내할 수 있다: 콘텐츠·과제·퀴즈·성적부·소통·분석·LTI 연동.
