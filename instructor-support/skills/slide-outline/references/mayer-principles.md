# Mayer 멀티미디어 학습 원리

Mayer(2009). 슬라이드·핸드아웃을 만들 때 **점검 규칙**으로 적용한다.

| 원리 | 내용 |
|---|---|
| **일관성 (Coherence)** | 학습과 무관한 단어·그림·소리를 제거한다 |
| **신호 (Signaling)** | 핵심에 강조 단서를 제공한다(슬라이드마다 강조 1개) |
| **공간 근접 (Spatial Contiguity)** | 관련된 글과 그림을 가까이 배치한다 |
| **중복 회피 (Redundancy)** | 화면 텍스트를 그대로 다시 읽어주지 않는다(발표자 노트는 부연·예시·발문) |
| **분절 (Segmenting)** | 소화 가능한 단위로 나눈다 |
| **사전훈련 (Pre-training)** | 핵심 개념·용어를 미리 제시한다 |
| **개인화 (Personalization)** | 형식적 문체보다 대화체가 더 깊은 학습을 유도한다 |

## 슬라이드 작성 규칙
- 한 슬라이드 **한 메시지**, 텍스트 최소화(불릿 3~5개, 각 12자 내외 권장)
- 핸드아웃은 능동 처리(빈칸 채우기형 guided notes·핵심 질문)를 유도
- 발표자 노트는 화면을 그대로 읽지 않고 부연·예시·발문으로 구성(중복 회피 + 개인화)

## 실행층 원칙
- 생성 콘텐츠가 **차시 학습목표를 지지**하는지 확인. 슬라이드·핸드아웃은 **새 목표를 만들지 않고** 기존 목표를 지지한다.
- 강의 노트를 학습자용 핸드아웃으로 변환할 때 '쉬운 말 + 예시'를 반영한다.
