# ABCD 학습목표 모델

Heinich et al.(1996). 학습목표는 교수자의 의도가 아니라 **학습자의 도달점**("학생이 무엇을 할 수 있다")을 관찰·측정 가능한 형태로 기술한다.

| 요소 | 의미 | 예 |
|---|---|---|
| **A** (Audience, 대상) | 목표의 주체가 누구인지 | "학생은 ~할 수 있다" |
| **B** (Behavior, 행동) | Bloom 동사로 관찰 가능한 행동 | "A/B 테스트 결과를 해석한다" |
| **C** (Condition, 조건·선택) | 어떤 조건에서 이루어지는지 | "프로젝트 관리 소프트웨어를 사용하여" |
| **D** (Degree, 수준·선택) | 도달 수준 | "90% 정확도로" |

## 측정 가능성의 원칙
- 관찰·측정 가능한 **단일 동사**로 기술한다.
- 측정 불가능한 동사('이해한다·배운다·안다·감상한다')는 쓰지 않는다.
- 한 목표에 동사는 하나만.

## 작성 예시
- ✅ "학생은 두 집단의 평균 차이를 t-검정으로 **검정하고**[적용], 결과를 유의수준 기준으로 **판단할**[평가] 수 있다." → 동사 2개이므로 **목표 2개로 분리**해야 함
- ✅ "학생은 A/B 테스트 데이터를 주면(C) 통계적 유의성을 **판정할**[평가] 수 있다(B)."
- ❌ "학생은 통계를 **이해한다**." (측정 불가 동사)

각 목표 뒤에는 `[기억]/[이해]/[적용]/[분석]/[평가]/[창조]` 인지수준 태그를 붙인다.
