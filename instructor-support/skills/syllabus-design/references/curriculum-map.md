# 커리큘럼 맵 (정렬 매트릭스)

강좌 목표가 **어느 주차에서 다뤄지고 어느 평가에서 측정되는지**를 한눈에 추적하는 표. 목표–주차–평가의 정렬을 가시화하고, 빠진 연결(다뤄지지 않는 목표, 어떤 목표도 지지하지 않는 주차)을 드러낸다.

## 형식
- **행**: 강좌 학습목표(G1, G2, …)
- **열**: 주차(W1…Wn)와 평가 항목(중간·기말·과제 등)
- **셀**: 해당 목표가 그 주차/평가에서 다뤄지거나 측정되면 ● 표시(도입 △, 심화 ● 등으로 구분 가능)

## 예시

| 목표 | W1 | W2 | W3 | … | 중간 | 기말 | 프로젝트 |
|---|---|---|---|---|---|---|---|
| G1 [이해] | ● | ● | | | ● | | |
| G2 [적용] | | ● | ● | | ● | ● | ● |
| G3 [분석] | | | ● | | | ● | ● |
| G4 [평가] | | | | | | ● | ● |

## 점검 규칙
- **모든 강좌 목표**가 최소 한 주차에서 다뤄지고 최소 한 평가에서 측정되는가?
- 어떤 목표도 지지하지 않는 주차가 있는가?
- 목표의 인지수준과 그 목표를 측정하는 평가의 인지수준이 일치하는가? (G3=[분석]인데 측정 평가가 암기형이면 정렬 오류)
- 주차 목표는 강좌 목표에서 분해되었는가(수준은 강좌 목표와 같거나 낮게)?
